### Key Highlights

#### 1. Amazon SageMaker - End-to-end ML Platform

- **Data preparation and labeling:** Preparing and labeling data for ML
- **Model training, tuning, and deployment:** Training, tuning, and deploying models
- **Integrated MLOps capabilities:** Built-in MLOps features
- **Live Demo:** SageMaker Studio walkthrough

#### 2. Generative AI with Amazon Bedrock

- **Foundation Models:**

  - Claude, Llama, Titan - comparison and selection guide
  - Understanding pros and cons of each model

- **Prompt Engineering:**

  - Techniques and best practices
  - Chain-of-Thought reasoning
  - Few-shot learning

- **Retrieval-Augmented Generation (RAG):**

  - RAG Architecture
  - Knowledge Base integration
  - Combining proprietary data with LLMs

- **Bedrock Agents:**

  - Multi-step workflows
  - Tool integrations
  - Automating complex tasks

- **Guardrails:**

  - Safety and content filtering
  - Controlling AI output

- **Live Demo:** Building a Generative AI chatbot using Bedrock
