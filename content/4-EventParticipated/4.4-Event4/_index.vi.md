### Nội Dung Nổi Bật

#### 1. DevOps Mindset & Culture

- **DevOps culture and principles:** Văn hóa cộng tác giữa Dev và Ops
- **DORA Metrics:** Đo lường hiệu quả DevOps
  - Deployment Frequency
  - Lead Time for Changes
  - Change Failure Rate
  - Mean Time to Recovery (MTTR)

#### 2. AWS DevOps Services – CI/CD Pipeline

- **AWS CodeCommit:**

  - Git strategies: GitFlow, Trunk-based development
  - Source control best practices

- **AWS CodeBuild:**

  - Build configuration
  - Testing pipelines
  - Buildspec files

- **AWS CodeDeploy:**

  - Blue/Green deployments
  - Canary deployments
  - Rolling updates

- **AWS CodePipeline:**

  - Pipeline automation
  - Orchestration của CI/CD workflow
  - Integration với các services khác

- **Demo:** Full CI/CD pipeline walkthrough

#### 3. Infrastructure as Code (IaC)

- **AWS CloudFormation:**

  - Templates và Stacks
  - Drift detection
  - StackSets cho multi-account

- **AWS CDK (Cloud Development Kit):**

  - Constructs và reusable patterns
  - Multi-language support (TypeScript, Python, Java, C#, Go)
  - L1, L2, L3 Constructs

- **Demo:** Deploying với CloudFormation và CDK
- **Discussion:** Choosing between IaC tools

#### 4. Container Services on AWS

- **Docker Fundamentals:**

  - Microservices architecture
  - Containerization principles

- **Amazon ECR:**

  - Image storage và scanning
  - Lifecycle policies

- **Amazon ECS & EKS:**

  - Deployment strategies
  - Scaling và orchestration
  - Fargate vs EC2 launch types

- **AWS App Runner:**

  - Simplified container deployment
  - Fully managed service

- **Demo & Case Study:** Microservices deployment comparison

#### 5. Monitoring & Observability

- **CloudWatch:**

  - Metrics, Logs, Alarms
  - Custom dashboards
  - Log Insights queries

- **AWS X-Ray:**

  - Distributed tracing
  - Performance insights
  - Service map visualization

- **Best Practices:**
  - Alerting strategies
  - Dashboard design
  - On-call processes

#### 6. DevOps Best Practices & Case Studies

- **Deployment strategies:**

  - Feature flags
  - A/B testing
  - Canary releases

- **Automated testing:**

  - Unit, integration, E2E tests
  - CI/CD integration

- **Incident management:**

  - Runbooks
  - Postmortems (blameless)

- **Case Studies:** Startups và enterprise DevOps transformations

### Những Gì Tìm hiểu về Được

#### DevOps Fundamentals

- **Culture over tools:** DevOps là văn hóa, không chỉ công cụ
- **Automation first:** Tự động hóa mọi thứ có thể
- **Continuous improvement:** Cải tiến liên tục dựa trên metrics

#### Technical Skills

- **CI/CD Pipeline:** Xây dựng pipeline hoàn chỉnh với AWS services
- **IaC mastery:** Sử dụng CloudFormation và CDK hiệu quả
- **Container expertise:** Triển khai microservices với ECS/EKS
- **Observability:** Monitoring và tracing distributed systems

#### Career Development

- **DevOps career pathways:** Hiểu rõ các con đường phát triển
- **AWS certification roadmap:** Lộ trình chứng chỉ AWS liên quan DevOps

### Ứng Dụng Vào Công Việc

- **Implement CI/CD:** Áp dụng CodePipeline cho các dự án hiện tại
- **Adopt IaC:** Chuyển đổi infrastructure sang code với CloudFormation/CDK
- **Containerize applications:** Đóng gói ứng dụng với Docker và triển khai trên ECS
- **Setup monitoring:** Xây dựng observability stack với CloudWatch và X-Ray
- **Practice blameless postmortems:** Áp dụng văn hóa học hỏi từ incidents
