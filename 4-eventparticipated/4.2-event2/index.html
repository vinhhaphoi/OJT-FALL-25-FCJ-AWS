  • Event Name: Vietnam Cloud Day 2025: Ho Chi Minh City Connect Edition for Builders

Key Highlights

1. Completing a large-scale migration and modernisation with AWS

This session focuses on valuable lessons from thousands of enterprises that have migrated and modernized on-premises workloads to AWS:

  • Proven mental models: Thinking frameworks for systematic migration approach
  • Technical best practices: Optimal technical methods for the migration process
  • Modernization pathways: Various modernization paths depending on application characteristics
  • AWS migration accelerators: Tools and services to speed up the migration process
  • Techcombank case study: Real-world example of building a solid foundation and strategic roadmap

2. Modernizing Applications with Generative AI-Powered Tools (Amazon Q Developer)

This session explores how Amazon Q Developer transforms the Software Development Lifecycle (SDLC):

  • Agentic capabilities: Intelligent automation across AWS Console, IDE, CLI, and DevSecOps platforms
  • Code generation & quality: Accelerate code creation and improve code quality
  • Auto-documentation: Automatically generate comprehensive documentation
  • Unit test generation: Automatically generate unit tests to improve reliability
  • Intelligent collaborator: AI acts as an intelligent partner in development

3. Panel Discussion: Application Modernization

In-depth discussion on accelerating business transformation through application modernization:

  • Panelists: Experts from OCB, LPBank Securities, and Ninety Eight
  • Topics: Modernization strategies, real challenges and solutions from Vietnamese enterprises
  • Best practices: Sharing practical implementation experience

4. Transforming VMware with AI-driven cloud modernisation

Guide for organizations accelerating cloud adoption with VMware estates:

  • AWS Transform: How to migrate quickly, safely, and cost-effectively
  • Step-by-step playbook: Step-by-step guide for the migration process
  • Downtime-aware patterns: Patterns to minimize downtime
  • Modernization roadmap: Roadmap for modernizing to EKS, RDS, and serverless

5. AWS security at scale: From development to production

Explore how to enhance cloud security throughout the lifecycle:

  • Comprehensive security approach: Identification, prevention, detection, response, and remediation
  • Security-by-design: Integrate security from the development phase
  • GenAI for security: Use GenAI for security analysis and operations automation
  • Resilient architectures: Build resilient architectures against emerging threats

Key Takeaways

Strategic Migration Thinking

  • Modernize while you migrate: Don’t just “lift and shift”, but modernize simultaneously during migration
  • Phased approach: Approach in phases, don’t rush the entire system at once
  • Business-driven migration: Always start from business value, not technology

AI-Powered Development Tools

  • Amazon Q Developer: Powerful AI assistant supporting the entire SDLC
  • Agentic AI: Intelligent automation capabilities beyond simple code completion
  • Quality improvement: AI helps improve code quality, documentation, and testing

VMware Migration Strategies

  • AWS Transform: Tools and methods for migrating VMware enterprises
  • Post-landing modernization: After landing, continue modernizing to containers and serverless
  • Minimal downtime patterns: Patterns to minimize impact on business operations

Security Best Practices

  • Shift-left security: Integrate security from the earliest development stage
  • GenAI-enhanced security: Use AI for analysis and threat detection
  • Continuous security posture: Security is a continuous process, not a one-time effort

Application to Work

  • Apply migration best practices: Use mental models and pathways learned for migration projects
  • Try Amazon Q Developer: Integrate into development workflow to increase productivity
  • Implement security-by-design: Apply security practices from the design phase
  • VMware modernization: Reference playbook for VMware migration projects
  • Panel insights: Apply lessons learned from real Vietnamese enterprises
